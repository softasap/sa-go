sa-go
=====

[![Build Status](https://travis-ci.org/softasap/sa-go.svg?branch=master)](https://travis-ci.org/softasap/sa-go)


Installs basic go language setup



Example of use:

<pre>

     - {
         role: "sa-go"
       }

</pre>

Advanced:

<pre>


     - {
         role: "sa-go",
         go_version: 1.6 
       }


</pre>
